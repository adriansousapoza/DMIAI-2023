Pandelampe med blåt og hvidt lys
Du finder her på siden pandelamper som kan lyse med blåt lys. Se udvalget nedenfor.
Blå lys kan bruges til forskellige opgaver. Det er dog mest politi og myndigheder som bruger det blå lys. Det kan bruges til at se efter sved og blod, som vil fremstå sorte. Det vil være tydeligere, end hvis der bliver brugt hvidt lys.
Blåt lys kan også bruges af jægere, hvis de leder efter spor i skoven
