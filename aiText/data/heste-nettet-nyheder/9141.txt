DM Sport Islandshest lørdagens program
2012-07-07T08:48:50
Her er lørdagens program for DM Sport 2012 på Hedeland. Der er mange gode heste og ryttere som idag skal konkurrere og finalepladser søndag.   Cafeen byder på god mad og der er boder at handle i, vejret byder på sol, så kom ud på Hedeland og se vores dygtige danske ryttere kæmpe om de præstige fyldte placeringer:   Lørdag d. 7. Juli: 8:30 - 19:00 08.30 V1 - senior b-finale 09.00 F1 - junior/ungrytter 10.15 F1 - senior 13.45 T.1 - junior/ungrytte 16.00 P1 - 250 m pasløb, 3. + 4. gennemløb 17.15 V1 - ungrytter b-finale 18.00 P3 - 150 m pasløb, 3. + 4. gennemløb   Vel mødt på Hedeland
